Componentes del Modelo ER

1. Entidades Principales

Estudiante

  • ID (PK)
  • Nombre
  • Fecha Nacimiento

Asistencia

  • RegistroID (PK)
  • Fecha
  • Hora

2. Relaciones y Cardinalidades

Estudiante 1:N Asistencia

Un estudiante puede tener múltiples registros de asistencia

3. Atributos Clave

Simples

  • ID Estudiante
  • Fecha Registro

Compuestos

  • Dirección (calle, ciudad, CP)

4. Restricciones de Integridad

Clave Primaria

ID Estudiante debe ser único

Referencial

Asistencia requiere Estudiante existente
